🧱 Esquema Físico Detallado

Las 8 tablas núcleo modeladas en Supabase (PostgreSQL) con todos sus tipos de datos, constraints y claves.

👥

perfiles_usuario

Identidad del estudiante. Sus condiciones socioeconómicas y detalles demográficos vitales.

  • 🔑 id UUID
  • 🔗 user_id UUID
  • nombre VARCHAR(100)
  • apellido VARCHAR(100)
  • edad INTEGER
  • ciudad VARCHAR(100)
  • nivel_educativo VARCHAR(100)
  • condiciones_socioec... JSONB
  • created_at TIMESTAMPTZ
  • updated_at TIMESTAMPTZ
📋

cuestionarios

Catálogo versionado de las pruebas vocacionales disponibles, con banderas de estado activo.

  • 🔑 id UUID
  • nombre VARCHAR(200)
  • version VARCHAR(50)
  • descripcion TEXT
  • activo BOOLEAN
  • created_at TIMESTAMPTZ
  • updated_at TIMESTAMPTZ
❓

preguntas

El banco de ítems atado en cascada a un test, integra opciones dinámicas utilizando matrices JSONB.

  • 🔑 id UUID
  • 🔗 cuestionario_id UUID
  • texto TEXT
  • tipo VARCHAR(50)
  • orden INTEGER
  • categoria VARCHAR(100)
  • peso DECIMAL(3,2)
  • opciones JSONB
  • created_at TIMESTAMPTZ
📝

resultados

Tabla puente de métricas clave inmutables que guarda la respuesta en bruto del estudiante.

  • 🔑 id UUID
  • 🔗 perfil_usuario_id UUID
  • 🔗 cuestionario_id UUID
  • respuestas JSONB
  • perfil_vocacional JSONB
  • fecha_realizacion TIMESTAMPTZ
🏛️

instituciones

Las universidades e institutos técnicos de todo el ecosistema. Posee información de contacto oficial.

  • 🔑 id UUID
  • nombre VARCHAR(200)
  • tipo VARCHAR(100)
  • ciudad VARCHAR(100)
  • departamento VARCHAR(100)
  • direccion TEXT
  • telefono VARCHAR(50)
  • email VARCHAR(100)
  • sitio_web TEXT
  • costo_promedio INTEGER
  • activa BOOLEAN
🎓

programas

Las Carreras Académicas en sí. Define las condiciones, el tipo de rama y su huella JSONB de compatibilidad.

  • 🔑 id UUID
  • 🔗 institucion_id UUID
  • nombre VARCHAR(200)
  • tipo VARCHAR(100)
  • area_academica VARCHAR(100)
  • duracion, modalidad VARCHAR
  • descripcion, reqs... TEXT
  • costo_matricula INTEGER
  • perfil_compatible JSONB
  • activo BOOLEAN
📅

convocatorias

Separa la disponibilidad en el tiempo (fechas de adminisiones semestre I / II) del programa de estudios estático.

  • 🔑 id UUID
  • 🔗 programa_id UUID
  • nombre VARCHAR(200)
  • fecha_apertura DATE
  • fecha_cierre DATE
  • cupos INTEGER
  • informacion_adicional TEXT
  • activa BOOLEAN (AS STORED)
  • created_at TIMESTAMPTZ
🏆

recomendaciones

La salida final del Algoritmo Predictivo. Se liga al usuario a través del resultado y marca compatibilidad.

  • 🔑 id UUID
  • 🔗 programa_id UUID
  • 🔗 resultado_id UUID
  • compatibilidad DECIMAL(5,2)
  • razones TEXT
  • vista BOOLEAN
  • created_at TIMESTAMPTZ

⚡ Soft Deletes (UPDATE vs DELETE)

En el mundo real **casi nunca usamos DELETE**. Si una carrera se canceló en Brota, aplicamos un UPDATE para conservar registros estadísticos históricos.

UPDATE convocatorias 
SET activa = FALSE, informacion_adicional = 'Cancelada'
WHERE id = 'cc9f1f0...';

🔍 10 Consultas Vitales

Aprovechando SQL Avanzado (LIKE, BETWEEN, IN, IS NULL) para crear filtros predictivos, dashboards y motores de match.

Segmenta instituciones exactas a las ciudades que el estudiante elija.

SELECT nombre, tipo, ciudad FROM instituciones 
WHERE ciudad IN ('Bogotá', 'Medellín', 'Cali');

Permite al usuario encontrar carreras que contengan fragmentos de sus intereses.

SELECT nombre, modalidad FROM programas 
WHERE nombre LIKE '%Ingeniería%';

Filtra programas cuyos costos de matrícula se ajustan a un rango determinado.

SELECT nombre, costo_matricula FROM programas 
WHERE costo_matricula BETWEEN 1500000 AND 3000000;

Detecta universidades que olvidaron registrar teléfonos para forzar actualización.

SELECT nombre FROM instituciones WHERE telefono IS NULL;

Segmentación por edad colegial (15-17) y ciudad para enviar SMS publicitarios a regiones fijas.

SELECT nombre, edad FROM perfiles_usuario 
WHERE edad BETWEEN 15 AND 17 AND ciudad IN ('Bucaramanga', 'Cartagena');

Búsqueda de encuestas de diagnóstico que estén en fase '1.X'.

SELECT nombre, version FROM cuestionarios 
WHERE version LIKE '1.%' AND descripcion IS NOT NULL;

Filtra instituciones con correos `.edu.co` oficiales que aún no tienen web adjunta reportada. 

SELECT nombre, email FROM instituciones 
WHERE email LIKE '%@%.edu.co' AND sitio_web IS NULL;

Obtiene todas las fechas de inscripciones a abrir el próximo semestre en particular (H1 2026).

SELECT nombre, fecha_apertura FROM convocatorias 
WHERE fecha_apertura BETWEEN '2026-01-01' AND '2026-06-30';

Alerta. Busca carreras "Médicas/Exactas" a las que se olvidó indicar requisitos restrictivos en BD.

SELECT nombre FROM programas 
WHERE requisitos IS NULL AND area_academica IN ('Salud', 'Ciencias Exactas');

Cruza Usuarios y Recomendaciones para revelar Matches por encima del 90% (Excelente compatibilidad).

SELECT p.nombre, p.apellido, r.compatibilidad 
FROM recomendaciones r
INNER JOIN resultados rs ON r.resultado_id = rs.id
INNER JOIN perfiles_usuario p ON rs.perfil_usuario_id = p.id
WHERE r.compatibilidad BETWEEN 90.00 AND 100.00;
